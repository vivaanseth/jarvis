# Product

## Register

product

## Users

One person using Jarvis on their own MacBook while studying, coding, organizing projects, and moving between everyday desktop tasks. They need fast actions without surrendering privacy or control.

## Product Purpose

Jarvis is a private, local desktop command center that translates ordinary language into visible, bounded macOS actions. Success means the common command is faster than manual navigation, uncertain requests are clarified, and risky actions never happen by surprise.

## Brand Personality

Calm, capable, and discreet. Jarvis should feel attentive without pretending to be human or omniscient.

## Anti-references

No generic cyberpunk clutter, terminal cosplay, copied film assets, cartoon assistant, generic SaaS dashboard, fake productivity data, decorative glass everywhere, or theatrical AI language. The interface may evoke a cinematic aerospace command system through restrained cyan telemetry, precise radial instrumentation, and amber warning states.

## Design Principles

1. Show the interpretation before the consequence.
2. Keep powerful actions bounded and reversible.
3. Put the user back in flow with minimal ceremony.
4. Earn trust with plain language and real state.
5. Let macOS conventions do familiar work.

## Accessibility & Inclusion

Meet WCAG AA contrast, support VoiceOver and keyboard navigation, never communicate state by color alone, and respect both the system Reduce Motion preference and Jarvis's own reduced-motion setting.
